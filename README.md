# Ruuvi monitor
Lightweight data collector and webui for Ruuvi devices. The idea behind is that this runs fine on Raspberry PI 1B with a handful of Ruuvi tags.

## Usage
### Installation
```
git clone [this-repo]
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
### Running
```
python main.py
```
